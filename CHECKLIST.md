### 5. 🔑 API Ключ (опционально)

Если хочешь реальный AI:

1. **Получи ключ** на https://console.groq.com/
2. **Вставь в `.env`**:
```
VITE_GROQ_API_KEY=gsk_твой_ключ_здесь
```
3. **Перезагрузи сервер** - готово!

### 6. 🐛 Возможные проблемы

#### Если что-то не работает:

1. **Проверь консоль** (F12 → Console)
2. **Посмотри красные ошибки**
3. **Проверь что .env файл правильный**

#### Если чат не сохраняется:
- Очисти localStorage: `localStorage.clear()`
- Перезагрузи страницу

#### Если AI не отвечает:
- Без API ключа использует локальные советы (работают!)
- С ключом - реальный Groq AI
