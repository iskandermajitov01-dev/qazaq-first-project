# 🎯 БЫСТРЫЙ СТАРТ: Включи Groq AI за 2 минуты

## ⏱️ За 2 минуты к AI

### Шаг 1: Получи API ключ (30 сек)

1. Открой https://console.groq.com/keys
2. Нажми "Create API Key"
3. Копируй ключ (выглядит так: `gsk_abc123...`)

### Шаг 2: Добавь ключ в проект (30 сек)

1. Создай файл `.env` в корне папки проекта (рядом с `package.json`)
2. Напиши одну строку:
```
VITE_GROQ_API_KEY=gsk_твой_ключ_здесь
```
3. Сохрани файл

### Шаг 3: Перезагрузи сервер (30 сек)

1. Закрой `npm run dev` (Ctrl+C в терминале)
2. Запусти заново: `npm run dev`
3. Готово! 🎉

---

## ✅ Проверь что работает

1. Открой http://localhost:5173
2. Нажми на вкладку **"AI Асс."** (в меню)
3. Напиши: **"Как начать с robotics?"**
4. Ждёшь ответ (2-5 сек)
5. ГОТОВО! 🚀

---

## 🆘 Если что-то не сработало

### Ошибка: "VITE_GROQ_API_KEY is not defined"
→ Проверь что файл `.env` существует и там правильное имя переменной

### Ошибка: "401 Unauthorized"
→ Ключ неправильный. Получи новый на https://console.groq.com/keys

### Всё молчит / нет ответа
→ Проверь консоль браузера (F12 → Console)
→ Посмотри красные ошибки

### Всё равно не работает
→ Используй локальные советы (они тоже неплохие!)
→ Без API ключа AI даст контекст-зависимые советы
